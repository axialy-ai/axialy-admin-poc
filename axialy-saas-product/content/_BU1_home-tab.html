Welcome to AxiaBA!

AxiaBA is your comprehensive business analysis control center and content generator, powered by generative AI to simplify and streamline your analysis activities. This web application provides all the tools you need to capture inputs, generate structured data, collaborate with stakeholders, and refine your business analysis artifacts over time.

How to Use AxiaBA

After logging in, you will land here, on the Home tab. The main workspaces you will use are: Generate, Refine, Reports, and Settings. Below is an overview of each tab’s purpose, along with guidance on how to use them step-by-step.

1. Generate Tab

In the Generate tab, you will create or build out your initial business analysis data. This is your launching point for capturing new ideas, concepts, or scenarios that AxiaBA can help you structure and expand.

  • Input Text: In the first ribbon, enter the textual description of the business scenario or analysis context you want to explore. AxiaBA automatically counts your characters and can generate an Input Text Summary (e.g., Title, Summary, UTC Timestamp).
  • Choose AI Focus Areas: In the second ribbon, select the focus area templates you’d like AxiaBA to apply to your input text. These templates help tailor AI-generated outputs to different analysis viewpoints, such as requirements, user stories, or risk assessments.
  • Enhance with AI (Third Ribbon): Press Send Selected to fetch generative AI results based on your input text and the chosen focus areas. When results arrive, AxiaBA displays them in new “dynamic ribbon” grids at the bottom. Each grid corresponds to one focus area and can be modified or expanded by you (for example, adding rows, deleting rows, or editing cells).
  • Save Data: After reviewing and adjusting the AI-generated data in these dynamic ribbons, press Save Data to store them within an analysis package in AxiaBA’s database. Before finalizing, you can review and update the package header (title, summary, and description).

Once saved, this new analysis package is available for further refinement and stakeholder collaboration in the Refine tab. Note that you can create multiple packages by repeating this process with new or existing input texts and different focus areas.

2. Refine Tab

In the Refine tab, you review and enhance existing analysis packages. AxiaBA displays a set of package “tiles,” each with high-level metrics (ID, name, summary, stakeholder feedback indicators, etc.). Selecting a tile opens the corresponding package’s focus areas and records.

  • Open Package: Click the package tile to load its details. If you need to filter or locate specific packages, use the search box. You can also enable “Show Deleted” if you want to see previously removed or archived packages.
  • New Focus Area: From the Package Actions menu, you can create a new focus area with its own set of records. This expands the package’s scope and can lead to new focus-area versions once changes are saved.
  • Refresh Package Content: Reloads the package data in real time. Handy if you suspect new stakeholder feedback has arrived, or if you made changes in another tab or session.
  • Show Deleted Content: Toggles the visibility of any “soft-deleted” focus areas or records. Useful for restoring or auditing older data.
  • Edit Package Header: Revisits the package’s name, summary, or long description. Similar to the overlay you see when saving a new package in Generate.
  • Remove Package: Marks the entire package as deleted. You can restore it later by enabling “Show Deleted” and selecting “Recover Package.”

Each focus area within the open package has a Refine Data or Refine Activities menu providing options like:

  • Enhance Content (AI): Leverage generative AI to improve existing data or reorganize it based on new instructions.
  • Request Feedback: Send a “Content Review Request” link to specified stakeholders who can provide feedback (Approve/Revise, etc.) without needing their own AxiaBA account.
  • Compile Stakeholder Feedback (AI): Summarize and apply stakeholder input automatically, producing a refined version of the focus area.
  • New Record: Add individual property-value records to the focus area.
  • Remove Focus Area: Soft-delete the entire focus area’s data, which can be recovered later if needed.
  • Recover Versions: Restore a previous version of the focus area if you wish to revert changes or compare historical data.

You can also Export CSV for any focus area, downloading current records for offline review.

Overall, the Refine tab offers versioned edits, stakeholder collaboration, and flexible data manipulation, making it the heart of AxiaBA for ongoing analysis and maintenance.

3. Reports Tab

The Reports tab provides operational insights into your analysis process. Initially, the focus is on stakeholder feedback metrics. You can see which requests are outstanding, which have been addressed, and how many items were revised or approved.

  • Stakeholder Feedback Requests Report: Track all your feedback invitations, listing the stakeholder email, associated package, focus area, and response status. Filter the report by stakeholder, package, or date range as needed.
  • Pagination & Filtering: Long lists are broken into pages, with user-configurable “records per page.” Filters enable focusing on a subset of requests (e.g., show only “unresponded” items).

As AxiaBA evolves, more report types and advanced analytics will be added, giving you deeper visibility into your analysis artifacts, collaboration patterns, and overall usage trends.

4. Settings Tab

The Settings tab lets you manage custom organizations and set your Focus Organization. This is particularly helpful if you're handling multiple clients, departments, or business units.

  • Create Custom Organization: Provide a name, logo, and optional contact details. Any newly generated or refined analysis packages can be assigned to this organization to keep data separate.
  • Focus Organization: Use the dropdown to decide which organization's data you want to see in Refine. Packages tagged to other organizations remain hidden unless you switch focus.
  • Organization Cards: Each custom organization displays in Settings with relevant details. You can edit or replace the logo, update contact info, or remove the organization if it’s no longer needed.

Organizing data by client or department is easy with AxiaBA’s built-in organizational controls. Switch your focus organization at any time to view different sets of packages in the Refine tab.
